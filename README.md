# TERA Reforged — Launcher releases (public)

Public release mirror for the **TERA Reforged launcher** self-update channel.

This repository hosts only the published launcher binaries and Velopack update assets
(`Setup.exe`, full/delta `.nupkg`, `releases.win.json`) on its **Releases** page. The launcher's
in-app updater (Velopack `GithubSource`) reads from here. Source is maintained in the private
development repository; releases are published here by CI on a prefix-less SemVer tag.

No source code is kept in this repository.
